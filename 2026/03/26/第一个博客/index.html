实验题目：利用云平台搭建个人博客
实验学时：2 实验日期： 2026.3.20
实验目的：本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。
具体包括：构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。

硬件环境： 
联网的计算机一台

软件环境：
Windows or Linux

实验步骤与内容：
1、安装 Node.js、Git 环境，全局安装 Hexo 博客框架。
2、在本地创建并初始化 Hexo 博客项目，安装项目依赖。
3、配置博客基本信息、访问地址、主题及 GitHub 部署参数。
4、使用命令新建博客文章，在 Markdown 文件中编辑内容。
5、执行清理与生成命令，构建静态网站文件并本地预览。
6、通过 Git 将静态网页文件推送至 GitHub 仓库。
7、在仓库中开启 GitHub Pages 功能，完成博客线上发布。
8、后续通过修改文章、重新生成并推送文件实现博客更新。

结论分析与体会：
通过本次个人博客搭建实验，我不仅掌握了Node.js、Git、Hexo等工具的基本使用方法，也深刻体会到了理论与实践结合的重要性。实验初期，由于对环境配置和工具操作不熟悉，出现了多种报错，但通过逐步排查、查阅资料和反复尝试，最终解决了所有问题。
同时，我也认识到细节在实验中的关键作用：配置文件中的主题名称、部署分支、仓库地址等参数的微小错误，都会导致实验失败；博客内容编辑后，必须执行清理、生成、推送等完整流程，才能实现线上更新。此外，本次实验也让我了解了静态博客的搭建原理，掌握了个人线上展示平台的搭建方法，不仅提升了自身的实操能力，也为后续相关学习和实践奠定了坚实基础。
